# TtDBot

Telegram to Discord forward bot.

## Requirements

- Node.js `22.12.0` or newer
- Telegram API credentials
- Discord bot token

## Setup

1. Clone the repo.
2. Copy `config.yml.example` to `config.yml`.
3. Fill in the values.
4. Run `npm install`.
5. Run `npm run build`.

## Start

- Local: `npm start`
- Docker: `docker compose up --build`

## Development

- `npm run dev`
- `npm run dev:debug`
- `npm run start:ts`
